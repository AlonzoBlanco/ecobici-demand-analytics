# Ecobici Demand Analytics

This repository contains scripts and analytics for the Ecobici system in Mexico City (CDMX).

## Data

The raw data files and processed database files are too large to be hosted on GitHub and are therefore excluded from this repository.

All raw data used in this project was obtained from the official Ecobici open data portal:
**[https://ecobici.cdmx.gob.mx/datos-abiertos/](https://ecobici.cdmx.gob.mx/datos-abiertos/)**

To run the scripts locally:
1. Download the trip data from the link above.
2. Place the raw files in the `data/raw/trips/` directory.
3. Run the processing scripts in the `scripts/` directory to build the database.
